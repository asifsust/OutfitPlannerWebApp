# Server

ExpressJS server

## Before developing
- Install nodejs
- Run following code to install dependences
    ```
    npm install
    ```

## Run and Access
- Run following code to run server
    ```
    npm start
    ```
- Go on the browser and go to
[localhost:8000](localhost:8000)
